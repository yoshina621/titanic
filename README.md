# このリポジトリーの目的
kaggleのTitanic - Machine Learning from Disasterを解くプロセスの可視化を行うことが目的です。
備忘録的な側面を持ちつつ、問題解決の言語化能力の向上を図りたいと思います。

approah.mdにこの問題を解くプロセスを記載しているので、気になる方はご覧ください。

# ファイル構造
kaggleのタイタニック問題に関するリポジトリです。
内容としては、下記が含まれています。
* data(学習や検証に使用するデータ)
* src(ソースコード)
* models(モデル)
* output(実行結果)
* tests(??)
* aproach.md(本問題の解決手順を記載したテキスト)
