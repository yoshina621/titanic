# 課題の理解
この課題(Titanic - Machine Learning from Disaster)は、タイタニック号に乗っていた人々の特徴から、「誰が生き残るか？」を予測するものである。
トレーニングデータには、一人ひとりの特徴と、生き残ったかのラベルが記載されている。
テストデータには、一人ひとりの特徴がありますが、生き残ったかどうかのラベルは記載されていない。
良いモデルをトレーニングデータを使って訓練し、訓練したモデルでテストデータのラベルを予測する。

# データのEDA
まず、データのEDA(Exploratory Data Analysis: 探索的データ分析)を行う。
EDAとは、データの分布、平均、外れ値、パターンなどを確認することであり、仮説形成のために分析をすることが目的である。
「まずは生データと仲良くなろう。詳しい話はそのあとだ。」、ということである。
